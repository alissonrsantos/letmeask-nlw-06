## Como executar
Clone o projeto e acesse a pasta do mesmo.
```bash
$ git clone https://github.com/alissonrsantos/letmeask-nlw-06
$ cd letmeask-nlw-06
```
Para iniciá-lo, siga os passos abaixo:
```bash
# Instalar as dependências
$ yarn 

# Iniciar o projeto
$ yarn start
```
